# Contributing to Nexus

> English translation in progress. See [docs/fr/CONTRIBUTING.md](../fr/CONTRIBUTING.md) for the full guide.

## Quick summary

- **Plugins & themes** → contribute freely in `nexus-plugins/` and `nexus-themes/`
- **Core code** → open an Issue first, then a Pull Request
- **Translations** → copy `docs/fr/` into `docs/{your-lang}/` and translate
- **Bugs** → open an Issue with steps to reproduce

## Commit format (required)

Follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat: add voice channel mute shortcut
fix: correct JWT expiry check
docs: update installation guide
refactor: extract voice signaling to separate module
test: add auth middleware unit tests
chore: update dependencies
```

All commit messages and code comments must be in **English**.

## Code of conduct

Critique ideas, not people. Build something good.
